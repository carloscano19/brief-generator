# SEO/GEO Brief Generator Pro

AI-powered content brief generator optimized for Search Engine Optimization (SEO) and Generative Engine Optimization (GEO).

**Live Demo:** [https://carloscanofernandez.com/brief-generator/](https://carloscanofernandez.com/brief-generator/)

## Features
- **4-Step Wizard:** Configuration -> Seed Keywords -> Selection -> Brief.
- **LLM Support:** OpenAI, Anthropic, and Google Gemini.
- **Streaming UI:** Real-time brief generation.
- **Exports:** DOCX and PDF support.
- **Responsive Design:** Premium mobile-first UI.

## Local Development
```bash
npm install
npm run dev
```

## Deployment
Automated via GitHub Actions to Hostinger (SSH/rsync).
